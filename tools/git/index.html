# git

TIP

Git是一个免费的开源 分布式版本控制系统，旨在快速高效地处理从小型到大型项目的所有内容。

Git 易于学习， 占地面积小，具有闪电般的快速性能。它具有诸如Subversion，CVS，Perforce和ClearCase之类的SCM工具，具有廉价的本地分支，便捷的暂存区域和 多个工作流等功能。
