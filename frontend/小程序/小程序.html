# 小程序单位rpx

  • 适配移动设备的一个单位
  • 是由小程序自己提供的
  • 设计稿宽度默认为 750px
  • 将来在写小程序时，可以使用 rpx 尽量使用 rpx
  • 部分情况下不建议使用 rpx:
    • 1.0 字体大小
    • 2.0 大小比较小的边框： border: 1rpx solid #ccc;
  • 总结：
    • a. 设计稿宽度为 750px
    • b. 直接去设置稿上量容器的尺寸，直接将数字设置，单位改为 rpx 就可以了

# 小程序中的横向滚动问题

1.scroll-view 中的需要滑动的元素不可以用 float 浮动；

2.scroll-view 中的包裹需要滑动的元素的大盒子用 display:flex; 是没有作用的；

# ---pormise---

作用:回调函数

回调函数的多层嵌套

好处:由于JS代码会应用到大量的回调函数,在回调函数嵌套的过程中会出现回调地狱,对代码扩展和维护不便,可以使用promise来解决

# 如何使用pormise

基本用法:

  • 是es6中提供的一个新的对象(构造函数)

  • 使用前需要new一下

  • 传入两个参数(函数): resolve():成功时执行的函数, reject():失败后执行的函数

    let pormise = new Promise((resove,reject)=>{
	//处理自己的逻辑代码
	//成功时执行的代码
	if(成功){
		resolve(data)
	}else{
		//失败后执行的代码
		reject(err)
	}
})
//执行逻辑代码
function resolve(data){ //调用resolv函数
	console.log(data.toString())
}

function reject(data){ //调用reject函数
	console.log(err)
}

pormise.then(resolve,reject)


//改成箭头函数
pormise.then(data) => {console.log(data.toString()),(err)=>{console.log(err)}}

    步骤:

    创建pormise对象

    let pormise = new Promise((resove,reject)=>{})

    执行逻辑代码

    pormise.then(resolve,reject)

# 了解 promise 的执行特点：

  • promise　有自己的状态：
    • pending：等待时
      • 当 promise 被创建时
    • fulfilled： 成功时
      • 当执行 resolve 时
    • rejected： 失败时
      • 当执行 reject 时
    • 总结：
      • resolve：
        • 修改 promise 的状态
        • 调用 then 中第一个方法
      • reject:
        • 修改 promise 的状态
        • 调用 then 中第二个方法
  • promise 一旦声明，会立即执行 promise 中的逻辑代码
    • 面试题
  • promise 后面可以接多个 then 方法
    • promise 中 then 方法的返回值
      • undefined
        • 下一个 then 中参数为 undefined
      • 普通数据
        • string
        • number
        • array
        • object
        • ....
        • 下一个 then 中的参数为 普通数据
      • promise 对象
        • 下一个 then 方法由这个 Promise 对象来调用

# 补充 API：

  • then: 执行 promise 的逻辑
  • catch:
    • 用来捕获 promise 中的错误
  • all（必须要所有）：
    • 将多个 Promise 实例，包装成一个新的 Promise 实例
      • 全部状态为 成功 最终才是 成功
      • 只要有一个状态为 失败 最终结果为失败
  • race（只需要其中一个）:
    • 将多个 Promise 实例，包装成一个新的 Promise 实例
      • 只要有一个的状态为成功，最终结果就为成功

# 使用promise封装wx.request

//封装一个wx.request 使用promise
function myRequest(options) {
    //判断是否存在 baseUrl
    options.baseUrl = options.baseUrl || "放请求地址";
    return new Promise((resolve,reject) => {
        //发送请求
        wx.request({
            url:options.baseUrl + options.url,//请求路径
            method:options.method || "GET",//请求方式(规范大写 GET POST)
            data:options.data || {},//请求参数
            header:options.header || {},//请求头
            success:function(res) {
                //成功后回调的函数
                resolve(res)
            },
            fail:function (err) {
                reject(err)
            }
        })
    })
}

//暴露方法
export default myRequest;
